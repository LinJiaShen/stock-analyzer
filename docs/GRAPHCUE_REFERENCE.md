# Graphcue 參考分析與引入計畫

> 來源：https://stock.cleanrich.com.tw（Graphcue 交易自動化模組）
> 分析日期：2026-06-11

## 觀察到的核心概念

### 1. 策略流程漏斗（Trading Flow）

每個「策略」是一條獨立的流程，股票在四個階段間流動：

```
候選列表 (Screening)  →  觀察列表 (Watchlist)  →  持倉列表 (Positions)  →  跟單帳戶 (Followers)
   每日篩選符合條件        符合觀察條件自動加入       模擬/實際進場單           多帳戶跟單執行
```

- **候選列表**：每日掃描全市場，列出符合策略條件的股票 + 技術形態旗標
- **觀察列表**：記錄「加入日期、加入價格、最新價格、漲跌幅、加入原因、停止觀察日期/原因」，分「觀察中 / 已停止」
- **持倉列表**：模擬單完整生命週期（進場 → 分批停利/停損 → 平倉）
- **跟單帳戶**：將策略訊號發送到多個帳戶（我們先不做）

### 2. 波動度策略矩陣

策略命名 = `期間 × 波動度`，例如：短期高波動、中期極波動 (<30)、長期低波動。

篩選條件組合（從「中期極波動 (<30)」副標題逆向）：
- `ATR14% > 6%`（波動度門檻，ATR 佔股價百分比）
- `ADTV 10日 ≥ 1萬張`（流動性門檻，10 日均量）
- `價格 < 30`（價格上限，控制資金門檻）

**我們的波動度分級定義**：
| 分級 | ATR14 / 收盤價 |
|------|---------------|
| 低波動 | < 3% |
| 中波動 | 3% – 6% |
| 高/極波動 | > 6% |

**期間對應突破回看窗口**：短期 = 20 根、中期 = 60 根、長期 = 120/180 根。

### 3. N 根指標突破

獨立的指標型策略（60根/120根/180根指標、長期指標突破）：
- **N根高點突破**：收盤價突破「前 N 根 K 線最高點」（Donchian 通道突破）
- **N根動能突破**：收盤價突破前 N 根最高收盤 + 成交量 > 1.5 × 10日均量
- **形態旗標**：出現防守腳、出現異常點、築底跡象（布林值欄位，我們以可計算的近似實作：防守腳≈長下影線、築底≈低檔窄幅整理）

### 4. 模擬單（Paper Trading）結構

持倉表格欄位：

| 區塊 | 欄位 |
|------|------|
| 進場 | 時間、價格、數量（張） |
| TP1/TP2/TP3 | 時間、價格、數量（分批停利） |
| SL1/SL2 | 時間、價格、數量（分批停損） |
| 持倉資訊 | 持倉狀態（持倉中/部分平倉/已平倉）、剩餘數量 |
| 損益 | 已實現損益金額/報酬率、未實現損益金額/報酬率、總損益、總報酬率 |

統計卡：**勝率、平均盈虧比、EV（期望值）、總投入成本、已實現損益、總損益（含未實現）**

- EV = 勝率 × 平均獲利 − (1−勝率) × 平均虧損
- 未實現損益以「最近可用收盤價」計算（每日收盤後更新）

### 形態旗標近似演算法（後續實作參考）

- **防守腳**：最近 5 根內出現「下影線 ≥ 2×實體」且收在當根高半部、量縮
- **異常點**：單日量 > 3× 20日均量 且 漲跌幅 > 2×ATR
- **築底跡象**：60 根低點未再創新低 + 最近 20 根振幅 < 0.6×前 40 根振幅

### 模擬單資料模型

`paper_trades` 表：entry（時間/價/量）+ `exits` JSONB 陣列（每筆 = type tp/sl、seq、price、quantity、filled_time、filled_price），`status` = open/partial/closed。已實現損益由 filled exits 計算，未實現由最新收盤計算。數量單位 = 張（1張 = 1000 股）。
